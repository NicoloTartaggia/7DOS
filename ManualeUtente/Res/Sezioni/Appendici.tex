\subsubsection{D}

\paragraph{Datasource} \Spazio
\label{sec:datasource} 
Una datasource è una sorgente di dati in Grafana. Solitamente si tratta di un database. Grafana supporta nativamente:
	\begin{itemize}
	\item CloudWatch;
	\item Elasticsearch;
	\item Graphite;
	\item InfluxDB;
	\item Microsoft SQL;
	\item MySql;
	\item OpenTSDB;
	\item PostgreSQL;
	\item Prometheus;
	\item Stackdriver.
	\end{itemize}
